# DevOps Quote Generator

A simple Python app that generates a random DevOps/tech quote and deploys it as a static webpage via GitHub Actions CI/CD pipeline.

## Built for
**Cloudlem DevOps Internship — Week 3 Lab**

---

## Project Structure

```
devops-quote-generator/
├── app.py                        # Core logic — quote bank + HTML generator
├── test_app.py                   # Pytest test cases
├── requirements.txt              # Python dependencies
├── Dockerfile                    # Container definition
├── index.html                    # Auto-generated by app.py (GitHub Pages)
└── .github/
    └── workflows/
        └── ci.yml                # Full CI/CD pipeline
```

## CI/CD Pipeline

```
Push to main
     ↓
Job 1: Run Tests (pytest)
     ↓
Job 2: Build & Push Docker Image to Docker Hub
Job 3: Generate index.html → Deploy to GitHub Pages
```

## How to Run Locally

```bash
pip install -r requirements.txt
python app.py         # Generates index.html
pytest -v             # Runs all tests
```

## GitHub Secrets Required

| Secret | Description |
|---|---|
| `DOCKER_USERNAME` | Your Docker Hub username |
| `DOCKER_PASSWORD` | Your Docker Hub password or access token |
